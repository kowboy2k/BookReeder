Comment ça marche

  • Lecture rapide — les mots défilent un à un ou par groupes, centrés sur la lettre repère (rouge) pour garder l'œil fixe.
  • Dialogues dynamiques — rajoutez du style dans les dialogues entre les personnages !
  • Mode Minimaliste — touchez la zone de lecture pour masquer l'interface (lecture sans distraction) ; touchez à nouveau pour revenir.
  • Mode loupe — appui prolongé sur le bouton lecture/pause pour afficher la lecture contextuelle + annotations.
  • Lecture horizontale — basculez la position des boutons en restant appuyé sur les flèches gauche/droite.
  • Naviguer — touchez les infos de lecture pour ouvrir la navigation et aller à un chapitre ou se déplacer dans le livre. En Mode Loupe, touchez la ligne d'infos du chapitre pour naviguer sans quitter la loupe.
  • Chapitres — les boutons ⏮ / ⏭ sautent au chapitre précédent / suivant ; les flèches reculent / avancent d'une phrase.
  • Vitesse — les boutons − et + ajustent la vitesse de lecture.
  • Accélération — appuyez sur la zone de vitesse et poussez vos limites en accélérant automatiquement !
  • Bibliothèque — vos livres restent enregistrés et reprennent à la bonne page, même hors-ligne.
  • Astuce — pour un meilleur confort, enregistrez la page sur l'écran d'accueil.
